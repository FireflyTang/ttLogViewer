# ttLogViewer

一个用 C++ 编写的快速高效的终端日志文件查看器，专为查看和过滤超大日志文件设计。

## 特性

- **大文件支持**：内存映射（mmap）+ 行索引，O(1) 跳转任意行，无需完整加载
- **编码自动识别**：通过 BOM 自动识别 UTF-8 BOM / UTF-16LE / UTF-16BE，解码后正常显示，无乱码
- **链式过滤**：多个过滤器顺序叠加（字符串或正则模式），每层缓存中间结果；编辑过滤器时 `Tab` 切换正则/字符串模式（与搜索逻辑统一），过滤栏显示每个过滤器的匹配行数及颜色指示圆点（⬤ 启用 / ○ 禁用，均显示过滤器颜色，label 选中高亮，圆点保持彩色）；过滤窗格按 `x` 跳转对应原始行
- **水平滚动**：`←`/`→` 键水平平移当前窗格内容，查看超出终端宽度的行
- **鼠标支持**：滚轮滚动焦点窗格（每次 1 行，Ctrl+滚轮透传终端用于字体缩放）；左键单击切换焦点并定位到点击行；**拖拽** 实现字符级文本选择，拖出窗格边界时自动垂直+水平滚动；`Ctrl+C` 复制到系统剪贴板（不含行号）；输入模式下 `Ctrl+V` 粘贴
- **实时追踪**：实时模式监控文件追加，`G` 键锁定跟随最新行；过滤超过 30 秒时弹出确认框
- **颜色标记**：每个过滤器对应独立颜色，匹配内容高亮显示；原始日志区不显示过滤器颜色
- **双区视图**：原始日志（60%高度）与过滤结果（40%高度）分区显示，两区独立滚动
- **会话持久化**：过滤链 + 上次打开文件自动保存，下次启动自动恢复
- **搜索**：焦点感知关键字搜索（原始区或过滤区），输入时 `Tab` 切换正则/字符串模式，`n`/`p` 前后跳转，当前结果关键词高亮显示（其他行不变）；`Esc` 或 `Tab` 切换窗格时清除搜索；底部状态行持续显示当前搜索词及结果计数
- **行号 & 折叠**：行号默认开启（右对齐，内容列固定），`l` 切换显示；`z` 折叠/展开超长行（折叠后颜色标记保留）
- **导出**：`w` 将当前过滤结果导出为文件
- **退出确认**：`q` 退出前弹出 Y/N 确认框，防止误操作
- **用户配置**：`~/.ttlogviewer.json` 可覆盖 UI 布局、搜索、Watcher 等参数

## 使用方法

```bash
ttlogviewer [文件路径]
```

进入后按 `h` 显示所有快捷键帮助。

## 构建

### 环境要求

- C++23 兼容编译器（GCC 13+、Clang 16+、MSVC 2022+）
- CMake 3.15+
- Ninja（推荐）

依赖项由 CMake FetchContent 自动下载，无需手动安装：
- [FTXUI v6.1.9](https://github.com/ArthurSonzogni/FTXUI)（TUI 框架）
- [nlohmann/json](https://github.com/nlohmann/json)（配置持久化）

### 构建步骤

**Linux / macOS**

```bash
cmake -B build -G Ninja
cmake --build build
```

**Windows（MSYS2 MinGW64）**

```bash
cmake -B build -G Ninja \
  -DCMAKE_C_COMPILER=C:/msys64/mingw64/bin/gcc.exe \
  -DCMAKE_CXX_COMPILER=C:/msys64/mingw64/bin/g++.exe
/c/msys64/mingw64/bin/ninja.exe -C build
```

编译产物：`build/bin/ttLogViewer`（Windows 为 `ttLogViewer.exe`）

## 快捷键速查

| 键 | 功能 |
|----|------|
| `h` | 显示帮助 |
| `↑` / `↓` | 上下移动光标 |
| `PgUp` / `PgDn` | 翻页 |
| `g` | 跳转到指定行号 |
| `G` | 跳转到末尾（实时模式下锁定跟随） |
| `Tab` | 切换原始 / 过滤 窗格焦点（同时清除当前搜索） |
| `←` / `→` | 水平滚动当前窗格（不超过最长行末尾） |
| `a` | 添加过滤器（输入时 `Tab` 切换正则/字符串模式） |
| `e` | 编辑当前过滤器（输入时 `Tab` 切换正则/字符串） |
| `d` | 删除当前过滤器 |
| `Space` | 启用 / 禁用当前过滤器 |
| `[` / `]` | 上 / 下选择过滤器 |
| `x` | 从过滤窗格跳转到当前行对应的原始行 |
| `/` | 搜索（输入时 `Tab` 切换正则/字符串模式，正则模式实时显示合法性圆点） |
| `n` / `p` | 跳转到下 / 上一个搜索结果 |
| `Esc` | 有活动选区时清除选区；有活动搜索时清除搜索；过滤进行中时中断过滤（按优先级依次触发） |
| 鼠标拖拽 | 字符级文本选择（跨行） |
| `Ctrl+C` | 复制选区到系统剪贴板（不含行号，折叠行复制完整内容） |
| `Ctrl+V` | 在输入模式（过滤器/搜索）粘贴剪贴板文本；多行或非文本剪贴板会弹错误对话框 |
| `s` | 切换静态 / 实时模式 |
| `o` | 打开文件（输入路径时 `Tab` 补全文件名/目录） |
| `l` | 切换行号显示 |
| `z` | 折叠 / 展开当前行 |
| `w` | 导出过滤结果 |
| `q` | 退出（弹出确认框） |

## 用户配置

启动时从以下路径加载可选 JSON 配置文件（文件不存在时静默跳过）：

- **Windows**：`%USERPROFILE%\.ttlogviewer.json`
- **Linux / macOS**：`$HOME/.ttlogviewer.json`

所有字段均可选，只需覆盖需要修改的值：

```json
{
  "uiOverheadRows":       6,
  "dialogMaxWidth":       60,
  "defaultTerminalWidth": 80,
  "watcherTickCount":     50,
  "watcherTickIntervalMs":10,
  "searchReserveFraction":10,
  "searchReserveMax":     10000,
  "jsonIndent":           2
}
```
